# ReCiclo

## Description
ReCiclo is a mobile app that promotes sustainability and reduces waste by connecting people who want to exchange objects they no longer need. Users can browse and search for items by category, and coordinate local pickup or delivery through in-app messaging.

## Architecture
The app follows the Clean Architecture pattern, allowing for a clear separation of concerns and greater scalability.

## Programming Language
* Flutter (Dart)

## Technologies Used
* Firebase (Authentication and Realtime Database)
* SQFlite (Local Database)
* Flutter Bloc (State Management)

## Features
* User registration and authentication
* Browse and search for items by category
* Post items for exchange
* In-app messaging for coordinating pickups and deliveries
* User profile with exchange history
